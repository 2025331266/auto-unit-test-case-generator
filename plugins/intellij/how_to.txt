To install the SmartUt plugin for IntelliJ, open the "Preferences" panel (e.g., from the toolbar).
On the left-side of the panel, choose "Plugins" and, on the right-side, click on
"Install Plugin from disk...".
Browse to were the SmartUtForIntelliJ.jar file was downloaded, and select it.
You will need to restart IntelliJ.
Be sure that the SmartUt plugin is activated, eg from the menubar
"File -> Other Settings -> Configure Plugins..."



To run SmartUt, just select one or more classes/packages in the "Project" view, right-click
and choose "Run SmartUt". Tests will be generated for all the selected classes/packages.
